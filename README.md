# Discworld MUD Client

Your own [Discworld](http://discworld.atuin.net) client in your browser.

## Features

- Command History
- Shield Tracking

## Installation & Usage

### Installation

    $ git clone git://github.com/dak1/discworld.git
    $ cd discworld
    $ npm install
    $ node server.js &
    $ open http://localhost:4242
